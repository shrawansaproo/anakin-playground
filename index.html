Configure Anakin API Key
Quick Setup
1
Go to app.anakin.ai → sign in
2
Navigate to Settings → API Keys → Create New Key
3
Copy the key (starts with sk-) and paste it below
4
Make sure node server.js is running — the proxy routes your request to Anakin
How it works: Your browser → localhost:3000/api/chat → api.anakin.ai
The Node proxy bypasses Anakin's browser CORS restriction. Key stored in localStorage.
